# Jaga's Data Structures

To implement different **Data Structures** In python.

## Instructions

1. Install:

```
pip install jds
```

2. How to use this library:

python code example:
```python
from jds.stack import Stack

stack = Stack()
stack.push(10)
stack.push(20)
stack.push(30)
stack.pop()
stack.display()

```

3. Study Docs to know more about each data structures
